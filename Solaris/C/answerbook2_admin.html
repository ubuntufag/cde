AnswerBook2

Cannot find local AnswerBook2 server to administer

There is no local AnswerBook2 server on this system. To administer a remote AnswerBook2 server either:

  • Enter the URL for the remote AnswerBook2 server. Then, when the server's Library Page appears, go to the Options Page and select the Administration GUI.

  • Log in on the system on which the remote AnswerBook2 server resides and bring up AnswerBook2 Admin from the CDE Front Panel.

Note: If administrative access control is turned on, you will be prompted to enter a valid administrative user name and password before the Administration GUI displays in the browser.
